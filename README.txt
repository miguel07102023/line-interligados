LINE INTERLIGADOS V8 — HOMOLOGAÇÃO

1. No Supabase SQL Editor, rode setup_v8.sql uma única vez.
2. Abra index.html e faça login com seu usuário real.
3. Cadastre 2 ou 3 contas de membro pela tela Criar conta.
4. Como Master, atribua as equipes na aba Membros.
5. Crie um evento.
6. Entre como cada membro e informe disponibilidade.
7. Volte como Master > Escalas > Gerar sugestão > Aprovar e publicar.
8. Entre como membro > Escalas > Confirmar ou recusar.
9. Confira notificações, dashboard e indicadores.

Para publicação:
- Suba esta pasta em um repositório GitHub e conecte na Vercel.
- Em produção, configure e-mail/WhatsApp/Google Calendar via APIs/Edge Functions.
